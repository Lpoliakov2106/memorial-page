# Национальное Вечное Хранилище

Сервис для создания и просмотра страниц памяти о близких людях.

**Демо:** [https://lpoliakov2106.github.io/memorial-page/](https://lpoliakov2106.github.io/memorial-page/)

---

## Возможности

- **Двухстраничная архитектура**
  - Главная страница со списком всех страниц памяти
  - Детальная страница профиля с полной информацией

- **Профиль человека**
  - Фото, ФИО, годы жизни, место
  - Полная биография
  - Timeline: рождение, учёба, карьера, семья

- **Поиск**
  - Поиск по имени на главной странице

- **Локальное хранение**
  - Все данные сохраняются в localStorage браузера
  - Работает без интернета после первой загрузки

- **Режим редактирования**
  - Включите режим редактирования для внесения изменений
  - Модальные окна для редактирования всех секций

## Локальный запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Lpoliakov2106/memorial-page.git
   ```

2. Откройте `index.html` в браузере:
   ```bash
   open index.html
   # или
   # python -m http.server 8000  (для локального сервера)
   ```

3. При первом запуске автоматически создаётся демо-страница.

## Использование

### Создание новой страницы памяти
1. На главной странице нажмите **"+ Создать новую страницу памяти"**
2. Вы перейдёте на новую страницу профиля
3. Нажмите **"Режим редактирования"** в правом верхнем углу
4. Заполните информацию, нажимая кнопки "Редактировать"

### Редактирование существующей страницы
1. Найдите страницу на главной и кликните по карточке
2. Включите **"Режим редактирования"**
3. Редактируйте нужные секции

### Поиск
- Введите имя в поле поиска на главной странице
- Нажмите "Найти" или Enter
