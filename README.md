# Страница памяти — MVP

Минимально-жизнеспособный прототип одностраничного сайта для создания цифровой страницы памяти человека. Разработан с акцентом на бережный тон, понятный UX и уважительное отношение к теме.

## О проекте

**Страница памяти** — это инструмент для родственников или ритуальных агентов, позволяющий создать достойную цифровую память о человеке. MVP включает базовые функции для:

- Хранения основной информации о человеке (имя, годы жизни, фото, цитата)
- Описания биографии и важных аспектов жизни
- Визуализации семейного дерева с близкими родственниками
- Сохранения данных локально в браузере
- Экспорта и импорта данных в формате JSON

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — адаптивная верстка, поддержка печати
- **Vanilla JavaScript** — без фреймворков и сборки
- **localStorage** — автоматическое сохранение данных

## Запуск

### Простой способ (без установки)

1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в браузере

Готово! Приложение полностью работает без сервера и сборки.

### Альтернативный способ (через локальный сервер)

Если вам нужен локальный сервер (например, для тестирования):

```bash
# Используйте любой простой HTTP-сервер
# Вариант 1: Python
python3 -m http.server 8000

# Вариант 2: Node.js (npx)
npx serve .

# Вариант 3: PHP
php -S localhost:8000
```

Откройте браузер по адресу `http://localhost:8000`

## Основные функции

### 1. Редактирование профиля

Нажмите кнопку **"Редактировать профиль"** в Hero-блоке, чтобы изменить:

- Имя человека (обязательное поле)
- Годы жизни (например: "1950–2024")
- Короткую цитату или послание
- Фото (укажите URL изображения или оставьте пустым)

### 2. Редактирование биографии

Нажмите **"Редактировать раздел"** в блоке "О человеке", чтобы заполнить:

- Основную биографию
- "Кем был(а)" — профессия, роли в семье
- "Что любил(а)" — хобби, интересы, любимые занятия
- "Главное, что важно помнить" — ключевые черты характера, воспоминания

### 3. Добавление родственников

Создайте семейное дерево, используя кнопку **"Добавить родственника"**.

Для каждого родственника можно указать:

- Имя (обязательно)
- Тип связи (обязательно): мать, отец, партнёр, сын, дочь, брат, сестра, внук, внучка, другое
- Годы жизни
- Фото (URL)
- Короткий комментарий

Кликните на узел в дереве, чтобы просмотреть, отредактировать или удалить родственника.

### 4. Хранение данных

Все изменения автоматически сохраняются в localStorage браузера. Данные сохраняются локально и не отправляются на сервер.

При первом запуске загружаются демо-данные с примером заполненной страницы.

### 5. Импорт и экспорт

**Экспорт JSON:**

Нажмите кнопку **"Экспорт JSON"**, чтобы скачать все данные в файл `memorial-page-export.json`. Используйте этот файл для резервного копирования или переноса на другое устройство.

**Импорт JSON:**

1. Нажмите **"Импорт JSON"**
2. Вставьте содержимое ранее экспортированного JSON-файла
3. Нажмите **"Импортировать"**

Данные будут восстановлены из файла.

### 6. Сброс демо-данных

Кнопка **"Сбросить демо-данные"** восстанавливает исходные демонстрационные данные. Используйте для тестирования или сброса всех изменений.

### 7. Режим просмотра

Переключатель **"Режим просмотра"** скрывает все управляющие элементы (кнопки редактирования, импорт/экспорт, сброс), оставляя только контент. Используйте этот режим для демонстрации страницы другим людям.

### 8. Печать

Кнопка **"Печать"** открывает диалог печати браузера. Печатная версия автоматически оптимизирована для формата A4:

- Скрываются все управляющие элементы
- Остаётся только основной контент
- Компактная вёрстка для размещения на 1–2 страницах

```json
      "loved": "Что любил(а)...",

```

## Roadmap — что дальше

Это MVP демонстрирует базовую концепцию. Вот возможные направления развития:

### 1. Гостевая книга

Возможность для друзей и близких оставлять воспоминания, слова поддержки и истории о человеке. Функции:

- Добавление записей с именем автора и датой
- Модерация записей
- Экспорт гостевой книги

### 2. Настройки приватности и защищённый доступ

Контроль доступа к странице памяти:

- Генерация защищённой ссылки с паролем
- Настройка видимости разделов (публичные/приватные)
- Журнал посещений

### 3. Совместное редактирование

Возможность для нескольких членов семьи редактировать страницу:

- Система ролей (администратор, редактор, гость)
- Синхронизация изменений между пользователями
- История изменений

### 4. Медиагалерея

Расширенная работа с медиафайлами:

- Галерея фотографий с описаниями и датами
- Видеозаписи и аудио (голос, любимая музыка)
- Документы и письма

### 5. Таймлайн жизни

Хронологическая визуализация важных событий:

- Рождение, образование, карьера, семья
- Достижения и памятные моменты
- Интерактивная временная шкала

### 6. QR-код для памятника

Интеграция офлайн и онлайн:

- Генерация QR-кода, ведущего на страницу
- Размещение на памятнике, табличке или в некрологе
- Статистика переходов по QR-коду

### 7. Интеграция с ритуальным агентом

Инструменты для ритуальных служб:

- Профиль агента на странице (контакты, услуги)
- Готовые шаблоны страниц памяти
- Конструктор для быстрого создания страниц
- Брендирование агента

### 8. Долгосрочное хранение и архивирование

Гарантия сохранности данных на десятилетия:

- Облачное резервное копирование
- Автоматическое продление хостинга
- Экспорт в долговременные форматы (PDF/A, HTML-архив)
- Интеграция с проектами цифрового наследия
